Steps to Run the Project

1. Extract the Project Files
Unzip the provided file and navigate to the pawfect_home directory.

2. Install Dependencies

Backend
Navigate to the backend folder: cd pawfect_home_back
Install the necessary packages: npm install

Frontend
Navigate to the frontend folder: cd ../pawfect_home_front
Install the necessary packages: npm install

3. Start the Application
Backend
Navigate to the backend folder: cd pawfect_home_back
Start the backend server: npm start

The backend will run on localhost:5001. (This project uses port 5001 due to MacOS port 5000 limitations.)

Frontend
Open a new terminal
Navigate to the frontend folder: cd pawfect_home_front
Start the frontend server: npm start
The frontend will run on localhost:3000.

Access the Application
Open http://localhost:3000 in your browser to view the application.
Ensure the backend server is running on http://localhost:5001 for proper functionality.

Notes
Open two terminals: one for the backend and one for the frontend.
Ensure ports 3000 (frontend) and 5001 (backend) are available before running the servers.
